# Agent Notes

- Security-sensitive correspondence, vulnerability reports, and coordinated disclosure questions go to `security@clawnexai.com`.
- All other project, support, legal, conduct, commercial, documentation, and general correspondence goes to `contact@clawnexai.com`.
- UI consistency rule: when two dashboard panels expose the same kind of workflow, keep command/action buttons in the same structural placement. Use one shared command-row pattern or the same two-row pattern across both; do not mix top-row commands in one panel with lower-card commands in another unless there is a deliberate product reason documented in the UI.
